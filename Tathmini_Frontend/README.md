# Tathmini Frontend

A React Native mobile application for the Tathmini project.

## Setup

1. Navigate to the TathminiApp directory:
   ```
   cd TathminiApp
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Run the application:
   - For Android:
     ```
     npm run android
     ```
   - For iOS (requires macOS):
     ```
     npm run ios
     ```
   - For web:
     ```
     npm run web
     ```

## Project Structure

The project was created using Expo, a framework and platform for universal React applications.

```
TathminiApp/
├── assets/         # Images, fonts, and other static assets
├── components/     # Reusable UI components
├── screens/        # Application screens
├── App.js          # Main application component
├── app.json        # Application configuration
└── package.json    # Project dependencies
```

## Development

- The application uses Expo for development
- To view the app on your phone, install the Expo Go app and scan the QR code displayed in the terminal
- For more information, refer to the [Expo documentation](https://docs.expo.dev/)
